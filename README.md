# Guid Typescript

Guid Typescript is a library that lets you generate guid code

## Installation and usage
### Installation

```
npm i guid-ts2 --save
```

### Basic usage

```typescript
import { Guid } from "guid-ts2";

export class Example {
    public id: Guid;
    constructor() {
        this.id = Guid.create(); // ==> b77d409a-10cd-4a47-8e94-b0cd0ab50aa1
    }
}
```
